# 1. Cloner le projet
```bash
git clone https://github.com/HanaT-DS/streamlit_dashboard_v1.git
```
# 2. Entrer dans le dossier
```bash
cd streamlit_dashboard_v1
```
# 3. Lancer l'app
```bash
bash scripts/launch.sh 
```
ou 
```bash
./scripts/launch.sh
```
# 4. Ouvrir dans ton navigateur
```bash
http://localhost:8501
```
